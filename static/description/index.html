Partner Unique Email

Prevent duplicate email addresses in your contact database

This module ensures data integrity by preventing the creation of multiple partners with the same email address. Keep your contact database clean and avoid duplicates!

Features

  • Validates email uniqueness on partner creation
  • Validates email uniqueness on partner update
  • Case-insensitive email comparison
  • Clear error message showing the duplicate partner
  • Works with contacts and companies

How It Works

When you try to save a partner with an email that already exists in the system, the module will display an error message indicating which partner already has that email.

Example error:
A partner with email "john@example.com" already exists: John Doe

Installation

  1. Download and install the module
  2. No configuration needed - it works automatically!
